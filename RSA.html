This is my final project for Introduction to Computer Science. The purpose of this project is to show how public key cryptography works and this is not, by any means, plausible for real-world usage.

Generating primes for the keys.

I recently discovered that the sieve of Sundaram it's slightly faster than the sieve of Eratosthenes while still being easy to implement and being deterministic. This sieving method allows me to generate a list with all primes below a number and we'll pick a number at random from that list. The prime numbers that I'll generate are relatively small.
